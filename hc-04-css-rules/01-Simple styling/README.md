
# 1.Simple styling
I created 3 html files using an index file taken from the previous exercises to test the 3 methods to use css   
* inline
``` 
<p style="color:red;">This is a paragraph.</p>
```
* embedded
``` 
<style>
h1, h2, h3, h4, h5 {
 font-weight: bold;
 text-align: center;
}
a {
 color: #16c616;
}
</style>
<p style="color:red;">This is a paragraph.</p>
```
* external (in file css)
``` 
body {
  background-color: lightblue;
}
```
